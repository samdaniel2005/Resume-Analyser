Key Features
AI-powered resume parsing and candidate analysis.
Automated candidate scoring and ranking system.
Multi-category skill assessment framework.
Interactive recruitment analytics dashboard.
Candidate shortlisting and status management.

Data-driven hiring recommendations.

Backend: Node.js / Python APIs
AI & NLP: Resume parsing, skill extraction, candidate scoring algorithms
